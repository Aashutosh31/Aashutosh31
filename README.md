## 🧠 Featured Projects

### 🌐 WebChat — Global-Scale Social Communication Platform
A modern, multi-paradigm social network merging low-latency persistent messaging, Discord-style communities, Reddit-like forums, and ephemeral social feeds into a single ecosystem.

**Key Highlights:**
* ⚡ **Real-Time Mesh:** Distributed WebSocket architecture synchronized across instances via Redis Pub/Sub.
* 🎙️ **Live Media:** Multi-party WebRTC-based voice and video rooms powered by LiveKit.
* 🛡️ **Zero-Trust Security:** End-to-end payload encryption using the native Web Crypto API and strict JWT rotation.
* 🧠 **Ambient AI:** Embedded ARC Assistant using Mistral AI + Pinecone vector memory for contextual community support.

**Engineering Focus:**
* System design for horizontal scalability and high concurrency.
* Monolithic architecture mapped with isolated, decoupled service layers.
* Dockerized multi-container deployment pipelines.

**Tech Stack:** React, Node.js, MongoDB, Redis, Socket.IO, LiveKit, Tailwind CSS, Docker

🔗 **Architecture & Docs:** https://github.com/Aashutosh31/webchat-system-design
🌐 **Live Production:** https://webchat.qzz.io

### 🤖 ARC-AI — Autonomous Real-Time Voice Agent
A low-latency AI agent that streams responses, executes tools, and maintains long-term memory.

**Key Highlights:**
* ⚡ Real-time streaming via WebSockets (token-by-token responses).
* 🗣️ Interruptible voice responses (TTS + buffering pipeline).
* 🧠 RAG-based memory (vector embeddings + semantic recall).
* 🔧 Autonomous tool execution (web, APIs, external actions).

**Engineering Focus:**
* Designed low-latency response pipelines.
